# 📖 About

This repository contains my **15-Day Microsoft Excel Learning Journey**. Each day includes the concepts I learned, screenshots, demo videos, practice files, and a short summary. The goal is to build strong Excel skills for **Data Analytics, Business Analysis, MIS, Reporting, and Dashboard Development**.
